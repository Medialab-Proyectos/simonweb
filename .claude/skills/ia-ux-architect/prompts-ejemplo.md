# Ejemplos de uso

## Auditoría de IA

Audita esta pantalla desde arquitectura de información. Detecta problemas de organización, etiquetado, jerarquía, navegación y encontrabilidad. Devuélvelo en formato: hallazgo, evidencia, impacto, recomendación y prioridad.

## Creación de estructura

Necesito crear la arquitectura de información de una app de movilidad para usuarios y empresas. Define objetos del sistema, navegación principal, módulos, jerarquía, taxonomía y etiquetas sugeridas.

## Sitemap

Con base en estos requerimientos, crea un sitemap jerárquico claro para desktop y mobile. Identifica módulos principales, submódulos y rutas clave.

## Revisión de naming

Evalúa estas etiquetas de menú y reemplázalas por alternativas más claras, consistentes y centradas en el usuario.

## Optimización de dashboard

Reestructura este dashboard para separar indicadores, acciones, historial, configuración y administración sin mezclar contextos.

## Evaluación de flujo

Analiza este user flow y detecta dónde el usuario debe abrir otras pantallas para completar una tarea. Propón cómo reducir esa fricción desde arquitectura de información.
