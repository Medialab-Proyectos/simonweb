# Interface Flow Prompt

Diseña una interfaz original y usable.

## Instrucciones

1. Identifica la tarea principal del usuario.
2. Detecta dónde una solución convencional sería genérica.
3. Aplica al menos 4 patrones creativos.
4. Propón 3 rutas de interfaz.
5. Evalúa cada una por:
   - claridad
   - velocidad de uso
   - carga cognitiva
   - diferenciación
6. Selecciona la mejor.
7. Entrega:
   - concepto base
   - layout propuesto
   - navegación
   - componentes
   - estados
   - microinteracciones
   - riesgos y mejoras
