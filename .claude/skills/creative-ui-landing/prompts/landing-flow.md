# Landing Flow Prompt

Diseña una landing page diferenciada, no basada en plantilla.

## Objetivo

Convertir visitantes en acción clara.

## Instrucciones

1. Resume el problema de negocio.
2. Identifica riesgos de plantilla.
3. Selecciona entre 4 y 6 patrones de creatividad.
4. Genera 3 direcciones creativas:
   - A. segura
   - B. atrevida
   - C. híbrida
5. Evalúa las 3 opciones con foco en:
   - claridad
   - diferenciación
   - conversión
   - viabilidad
6. Elige una o combina dos.
7. Entrega:
   - concepto rector
   - estructura de secciones
   - dirección visual
   - CTA principal
   - componentes clave
   - por qué no parece plantilla
