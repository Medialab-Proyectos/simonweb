# Visual Accessibility

## Principios

- contraste suficiente
- foco visible
- targets interactivos razonables
- estados distinguibles
- no depender solo del color

## Revisiones mínimas

- textos clave legibles
- botones claramente identificables
- hover, focus y disabled distinguibles
- iconos con soporte visual suficiente
- mensajes de error y éxito comprensibles

## Riesgos frecuentes

- placeholders con bajo contraste
- textos secundarios casi invisibles
- links y botones visualmente ambiguos
- alertas diferenciadas solo por color
