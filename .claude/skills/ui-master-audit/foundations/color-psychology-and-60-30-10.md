# Color Psychology and 60-30-10

## Regla 60-30-10

Usar como principio orientador:

- 60% color dominante
- 30% color secundario
- 10% color de acento

No aplicarla de forma rígida. Ajustarla según densidad, branding y tipo de producto.

## Psicología del color

Interpretar con contexto, no como verdad absoluta.

### Azul

- confianza
- tecnología
- seguridad
- estabilidad

### Verde

- éxito
- confirmación
- salud
- avance

### Rojo

- error
- urgencia
- peligro
- alerta

### Amarillo / ámbar

- advertencia
- energía
- atención

### Negro / gris oscuro

- sofisticación
- fuerza
- premium
- alto foco

## Reglas de validación

Revisar:

- contraste entre texto y fondo
- rol claro de cada color
- consistencia entre estados
- abuso del color de acento
- uso innecesario de colores decorativos
- dependencia exclusiva del color para comunicar estado
