# Typography and Layout

## Tipografía

Evaluar:

- escala tipográfica
- coherencia de pesos
- legibilidad
- line-height
- longitud de línea
- contraste de jerarquías

## Buenas prácticas

- evitar demasiados tamaños sin sistema
- evitar demasiados pesos sin razón
- dar prioridad a lectura funcional
- usar el peso tipográfico como recurso jerárquico real

## Layout

Evaluar:

- sistema de columnas
- spacing por múltiplos consistentes
- padding interno
- separación entre grupos
- relación entre contenido y contenedor

## Grid

Preferir:

- sistema de 8pt o 4pt
- patrones repetibles
- consistencia entre secciones
