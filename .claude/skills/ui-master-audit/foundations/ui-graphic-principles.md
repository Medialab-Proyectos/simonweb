# UI Graphic Principles

## Principios esenciales

### Jerarquía visual

Siempre revisar:

- tamaño
- peso
- contraste
- posición
- proximidad
- repetición
- foco visual

### Composición

Evaluar:

- balance
- alineación
- espacio negativo
- ritmo
- consistencia de bloques
- densidad

### Gestalt aplicada a UI

Aplicar:

- proximidad
- similitud
- continuidad
- cierre
- figura-fondo
- región común

### Escaneabilidad

Preguntas:

- ¿La pantalla puede entenderse rápido?
- ¿Existen bloques claros?
- ¿Hay demasiados puntos compitiendo?
- ¿La acción principal se detecta sin esfuerzo?

### Economía visual

Favorecer:

- menos ruido
- menos decoración gratuita
- mayor claridad
- agrupación lógica
