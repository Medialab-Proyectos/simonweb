# UI Audit Checklist

## Jerarquía

- [ ] El título principal destaca
- [ ] La acción principal se identifica rápido
- [ ] El contenido secundario no compite en exceso
- [ ] Existe recorrido visual claro

## Layout

- [ ] Hay alineación consistente
- [ ] Los grupos están bien organizados
- [ ] El espacio en blanco aporta claridad
- [ ] No hay saturación innecesaria

## Color

- [ ] El color dominante está claro
- [ ] El color de acento no está sobreusado
- [ ] Los estados se diferencian bien
- [ ] El contraste es suficiente

## Tipografía

- [ ] Existe escala tipográfica
- [ ] Los pesos son coherentes
- [ ] La lectura es cómoda
- [ ] No hay tamaños arbitrarios

## Componentes

- [ ] Botones consistentes
- [ ] Inputs consistentes
- [ ] Cards consistentes
- [ ] Modales consistentes
- [ ] Alerts consistentes
- [ ] Tabs consistentes

## Accesibilidad visual

- [ ] Focus visible
- [ ] Contraste suficiente
- [ ] Elementos interactivos distinguibles
- [ ] No depende solo del color

## Consistencia sistémica

- [ ] La pantalla parece parte de un sistema
- [ ] Los patrones se repiten correctamente
- [ ] No hay estilos aislados
