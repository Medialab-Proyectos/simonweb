# Severity Matrix

## Crítico

Problema que afecta:

- comprensión primaria
- legibilidad
- acción principal
- confianza del producto
- accesibilidad visual clave

## Medio

Problema que:

- genera fricción
- afecta consistencia importante
- reduce claridad
- complica el escaneo
- puede provocar error o duda

## Deseable

Mejora que:

- aumenta madurez visual
- mejora consistencia fina
- mejora percepción de calidad
- optimiza detalle visual
