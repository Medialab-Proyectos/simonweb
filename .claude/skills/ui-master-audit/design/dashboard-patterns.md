# Dashboard Patterns

## Reglas

- destacar insights primero
- no saturar con métricas del mismo peso
- usar tarjetas solo cuando haya significado real
- separar métricas, acciones y detalle
- definir muy bien la acción principal

## Riesgos

- demasiados acentos visuales
- jerarquía plana
- bloques sin agrupación
- indicadores sin contexto
