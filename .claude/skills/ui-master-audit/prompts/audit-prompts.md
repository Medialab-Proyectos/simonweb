# Audit Prompts

## Prompt 1

Audita esta pantalla desde UI profesional. Evalúa jerarquía visual, composición, color, tipografía, spacing, componentes, consistencia y accesibilidad visual. Clasifica hallazgos en crítico, medio y deseable.

## Prompt 2

Analiza esta interfaz con enfoque en diseño gráfico aplicado a UI. Identifica si la regla 60-30-10 está bien balanceada, si el color guía correctamente la atención y si la interfaz comunica madurez visual.

## Prompt 3

Revisa esta pantalla como si fueras un director de UI. Señala qué funciona, qué rompe la consistencia y qué impide que la pantalla se sienta profesional.
