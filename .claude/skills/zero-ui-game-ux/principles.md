# Principios Zero UI / CXD

## Núcleo

- Toda acción humana nace en un estado de conciencia.
- El usuario no recorre flujos; transita estados.
- El comportamiento es consecuencia, no objetivo directo.
- La IA debe amplificar criterio, no amplificar error.
- Un sistema bien diseñado no decide por ti; te ayuda a decidir mejor.

## Capas

1. Fenomenología
2. Noética
3. Ethos
4. Contexto adaptativo

## Estados críticos a evaluar

- orientación
- carga cognitiva
- tiempo subjetivo
- seguridad y confianza

## Señales de deterioro UX

- microclics erráticos
- permanencia excesiva sin acción
- rebote interno
- abandono silencioso
- acción rápida seguida de arrepentimiento

## Patrones deseables

- orientación continua
- revelado progresivo
- transparencia funcional
- pausa consciente
- salida limpia
- salida digna
- explicabilidad de IA
- refuerzo interpretativo
- agencia conservada
- continuidad narrativa

## Antipatrones

- presión artificial
- confirmaciones opacas
- automatización sin consentimiento
- persuasión sin contexto
- cajas negras
- manipulación del impulso
- fricción burocrática para salir
