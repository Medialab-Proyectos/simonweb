# Trigger Examples

## Invoke the skill for prompts like:

- Audit this onboarding and explain why users may abandon it
- Improve this pricing page using UX psychology
- Review this checkout flow for decision friction
- Help me redesign this dashboard so users notice what matters
- Improve these CTAs so users understand the next step
- Suggest experiments to increase completion of this form
- Analyze this interface using UX-Core cognitive biases

## Do not invoke the skill for prompts like:

- Create the React component for this modal
- Convert this UI into Tailwind
- Make this screen look more modern
- Export these Figma tokens to JSON
- Write CSS for this layout
