# Ejemplos de mejora

## CTA genérico

Antes:

- Continuar

Después:

- Revisar solicitud
- Guardar y seguir
- Confirmar pago

## Error ambiguo

Antes:

- Algo salió mal

Después:

- No pudimos guardar los cambios. Revisa tu conexión e inténtalo de nuevo.

## Placeholder usado como label

Antes:

- Placeholder: Ingresa tu correo

Después:

- Label: Correo electrónico
- Placeholder: nombre@correo.com

## Estado vacío

Antes:

- No hay datos

Después:

- Aún no has agregado ningún método de pago.
- Agrega uno para completar tus compras más rápido.
- CTA: Agregar método de pago

## Confirmación vaga

Antes:

- Proceso exitoso

Después:

- Tu solicitud fue enviada.
- Te avisaremos por correo cuando haya una actualización.
