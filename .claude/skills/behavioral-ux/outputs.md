# Tipos de salida

## A. Auditoría ejecutiva

Ideal para stakeholders.
Formato:

- resumen
- top 5 hallazgos
- impacto
- quick wins
- oportunidades estratégicas

## B. Auditoría detallada

Ideal para diseño y producto.
Formato:

- hallazgo
- evidencia
- principio conductual
- impacto
- recomendación
- prioridad

## C. Propuesta de rediseño

Ideal para avanzar a wireframe/prototipo.
Formato:

- objetivo
- decisiones clave
- estructura del flujo
- componentes sugeridos
- copy sugerido
- métricas a validar

## D. Matriz impacto / esfuerzo

Ideal para roadmap.
Formato:

- cambio
- impacto esperado
- esfuerzo
- riesgo
- prioridad
