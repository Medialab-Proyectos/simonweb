# Decision tree

1. Is the user asking to design or to evaluate?
   - Design -> go to design mode
   - Evaluate -> go to audit mode

2. Is there a strong accessibility component?
   - Yes -> add ISO 9241-171

3. Is the task about requirements or acceptance criteria?
   - Yes -> add ISO/IEC/IEEE 29148 and ISO/IEC 25030

4. Is the user asking for metrics, KPIs, benchmark or validation?
   - Yes -> add ISO/IEC 25019, 25020, 25022, 25040

5. Is the interface natural, conversational, multimodal, gesture-based, or voice-based?
   - Yes -> add ISO/IEC 4944

6. Is the user asking about process, governance or operating model for UX?
   - Yes -> add ISO 9241-210, 220 and optionally 221
