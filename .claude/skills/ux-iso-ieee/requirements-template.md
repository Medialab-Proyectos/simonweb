## Acceptance criteria examples
- The system shall show the current step and total number of steps throughout the process.
- The system shall preserve all previously entered data when the user navigates backward.
- The system shall validate critical inputs before submission and explain how to recover.
- The system shall provide visible feedback after a user action within an appropriate response time.
- The system shall allow users to review and confirm irreversible actions before execution.
