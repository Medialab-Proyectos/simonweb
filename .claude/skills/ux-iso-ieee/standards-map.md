# Standards map

## Core UX and ergonomics

| Standard | Use it for | Typical outputs |
|---|---|---|
| ISO 9241-11 | Define or assess usability in context | usability analysis, UX scorecards, research plans |
| ISO 9241-110 | Interaction principles | heuristic audit, interaction review |
| ISO 9241-112 | Presentation of information | hierarchy, readability, scanability recommendations |
| ISO 9241-115 | UI design guidance | component decisions, flow structure, forms, navigation |
| ISO 9241-171 | Software accessibility | barrier list, remediation plan |
| ISO 9241-210 | Human-centred design process | discovery plan, workshop plan, design process checks |
| ISO 9241-220 | Organizational HCD process | UX governance, maturity, operating model |
