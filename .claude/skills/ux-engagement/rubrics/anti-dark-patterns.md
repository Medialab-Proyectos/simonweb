# Anti Dark Patterns

## Señales de alerta

- urgencia falsa
- escasez falsa
- opt-out escondido
- confirmshaming
- notificaciones excesivas
- premios que no aportan valor
- comparaciones sociales dañinas
- streaks que generan culpa excesiva
- pérdida artificial por no entrar
- bloqueo de funciones básicas para presionar
- copia confusa para inducir una acción

## Alternativas éticas

- transparencia
- control del usuario
- configuraciones visibles
- recompensa proporcional
- lenguaje claro
- recordatorios configurables
- pausas sin castigo
- progreso acumulativo recuperable
- comparación con progreso propio, no solo con terceros

## Regla de decisión

Si la mecánica aumenta uso pero disminuye autonomía, confianza o bienestar, debe rediseñarse.
