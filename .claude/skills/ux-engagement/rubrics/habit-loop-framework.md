# Habit Loop Framework

## 1. Trigger

Tipos:

- externo: push, email, recordatorio, banner, CTA, calendario
- interno: ansiedad, curiosidad, ambición, deseo de control, deseo de progreso

Preguntas:

- ¿qué dispara la acción?
- ¿llega en el momento correcto?
- ¿es relevante o solo intrusivo?

## 2. Motivación

Preguntas:

- ¿qué gana la persona?
- ¿qué dolor evita?
- ¿qué emoción activa la conducta?
- ¿qué tan visible es el beneficio?

## 3. Capacidad

Preguntas:

- ¿la acción es simple?
- ¿hay demasiados pasos?
- ¿requiere pensar demasiado?
- ¿hay barreras técnicas o cognitivas?

## 4. Acción

Preguntas:

- ¿cuál es la conducta mínima?
- ¿está claramente destacada?
- ¿hay una sola acción principal o demasiadas?

## 5. Recompensa

Preguntas:

- ¿qué recibe la persona de inmediato?
- ¿la recompensa es útil, emocional o social?
- ¿es suficientemente clara para impulsar repetición?

## 6. Inversión

Preguntas:

- ¿qué deja configurado para la próxima vez?
- ¿qué personaliza o construye?
- ¿cómo hace más probable el regreso?

## Lectura del loop

- Si falla trigger: el usuario no vuelve
- Si falla motivación: el usuario no ve valor
- Si falla capacidad: el usuario abandona
- Si falla recompensa: el usuario no repite
- Si falla inversión: el regreso no se consolida
