# Prompts sugeridos para usar este skill

## Auditoría general

Audita esta interfaz usando WCAG 2.2, ergonomía visual y ergonomía física.
Quiero:

- hallazgos priorizados,
- severidad,
- impacto en el usuario,
- recomendación puntual,
- criterio relacionado.

## Revisión de componente

Evalúa este componente desde accesibilidad y ergonomía.
Revisa legibilidad, contraste, foco, interacción táctil, claridad, carga cognitiva y facilidad de uso.

## Diseño desde cero

Ayúdame a diseñar esta pantalla aplicando principios de WCAG 2.2, ergonomía visual y física.
Quiero reglas concretas de layout, tipografía, contraste, targets, navegación, formularios y feedback.

## Revisión mobile

Evalúa esta pantalla mobile para uso con una mano, legibilidad, targets táctiles, jerarquía visual y fatiga visual.
Dame mejoras concretas.

## Criterios de aceptación

Convierte esta historia de usuario en criterios de aceptación UX/UI con enfoque en accesibilidad WCAG 2.2 y ergonomía visual/física.

## Revisión de formulario

Audita este formulario y detecta barreras de accesibilidad, claridad, esfuerzo físico y carga cognitiva.
Propón rediseño y microcopy de error.
